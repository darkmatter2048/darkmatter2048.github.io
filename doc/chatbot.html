首次使用需要配置一次环境

点击路径框输入cmd并回车，在弹出的黑框框中输入以下代码

python -m pip install --upgrade pip

5.进阶玩法

这个自动聊天程序的原理可以概括为：将张三发送给你的消息全都自动转发给AI小冰，再将AI小冰的回复全部自动转发给张三，你的微信在程序的帮助下成为了张三和AI小冰沟通的桥梁。

原理图：

但是这样可能会造成一个问题，那就是AI小冰在自我介绍的时候可能会说自己是小冰，你的聊天对象(张三)便知和ta聊天的并不是你本人，这不就穿帮了吗。

然而你可以通过修改文件夹根目录下的rules.json文件来修改AI小冰回复的内容进而避免穿帮。

单击rules.json文件并右键选择用记事本打开该文件会发现其中的内容如下：

{"小冰": "DaYe", "公司": "个人", "少女": "少年", "Xiao Bing": "DaYe"}

这些代码的意思是将AI小冰的回复中所有的 小冰替换为DaYe将公司替换为个人······。以此类推，你可以继续按照原来的格式续写这个文件，比如：

{"小冰": "DaYe", "公司": "个人", "少女": "少年", "Xiao Bing": "DaYe", "替换我": "换成我"}

如果你认为AI小冰的回复足够完美，不需要对其进行修改，可以直接把rules.json改成这样：

{"": ""}
